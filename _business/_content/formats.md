## Templates by Format

### LinkedIn Post Template
```
Hook (1 line that stops scroll)

Story or observation (3-5 short lines)

The insight/lesson

CTA
```

### Twitter Thread Template
```
1/ Hook
2-7/ Main points
8/ Summary + CTA
```

### Email Template
```
Subject line
Preview text

Opening hook

Body (short paragraphs)

CTA

P.S. (engagement driver)
```
